# Introduction
This is a starter boilerplate for React-native. Includes redux, redux-logger, redux-saga and fresco:gif to support gifs on android devices

## Installation

```
$ npm install
```

## Build ios

```
$ react-native run-ios
```

## Build android

```
$ react-native run-android
```
